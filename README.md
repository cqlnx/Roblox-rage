## Features

### Core Features
- **ESP** — Boxes, names, health bars, and distance on all players
- **Aimbot** — 360° auto-aim to closest visible player
- **Triggerbot** — Auto-fires when crosshair is on target
- **Hitmarkers** — Visual feedback on damage dealt

### Customization (F1 Menu)
- **FOV** — Adjust field of view (60-120)
- **Transparency** — Player model opacity (0-100%)
- **Spin Speed** — Character spin speed (0-100)
- **Color Picker** — Customize Glow, ESP, and Target colors with R/G/B sliders

### Visual Style
- HVH aesthetic with purple glow and transparent player model
- Purple lighting effects when enabled
- Clean splash screen on load

## Controls

| Key | Action |
|-----|--------|
| `F1` | Open/Close menu |
| `Right Shift` | Enable/Disable cheat |
| `ESC` | Close menu |
| `↑/↓` | Navigate menu |
| `←/→` | Adjust sliders / toggle values |
| `Enter/Space` | Toggle ON/OFF |
| `Mouse Click` | Click settings, drag sliders, open color pickers |

## Installation

1. Copy the entire script
2. Paste into your executor
3. Execute
4. Wait for splash screen to fade (3 seconds)
5. Press `Right Shift` to enable
6. Press `F1` to open menu and customize

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Script doesn't load | Check executor compatibility |
| Menu not showing | Press `F1` after splash screen |
| Aimbot not working | Ensure `Aimbot` is ON in menu |
| ESP not showing | Ensure `ESP` is ON in menu |
